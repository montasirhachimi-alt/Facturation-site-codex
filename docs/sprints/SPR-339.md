# SPR-339 — Quick Create Everywhere

Date: 2026-07-09

## Objective

Extend the Universal Command Center with Quick Create actions so users can start common business creation flows without first navigating to a workspace.

## Scope

- UI and product-experience layer only.
- Local client-side command center action routing only.
- No Runtime, Services, APIs, Prisma, database, permissions or business logic changes.
- No backend endpoints, AI, workflow automation or persistence changes.

## Action Registry Architecture

- Added a dedicated `ActionRegistry` for Quick Create actions.
- The registry is independent from:
  - `CommandCenterRegistry` for navigation.
  - `RecordSearchRegistry` for local business records.
- `getFoundationSearchSections` now composes groups in this order:
  - Quick Create
  - Navigation
  - Records
- The Command Center dialog remains generic and renders grouped `UniversalSearchSection` data.
- Quick Create items use `actionId` instead of `href`, allowing the shell to open a dialog rather than navigate.

## Supported Quick Create Actions

- New Company
- New Contact
- New Customer
- New Opportunity
- New Quote
- New Invoice
- New Payment

Each action includes English and French aliases such as `new`, `create`, `nouveau`, `devis`, `facture`, `paiement`, `contact`, `societe`, `quote`, `invoice` and `payment`.

## Dialog Experience

- Selecting a Quick Create item closes the Command Center automatically.
- The shell opens a creation dialog immediately.
- Existing domain dialogs are reused for:
  - Company
  - Contact
  - Customer
- Quote
- Opportunity, Invoice and Payment use the same shared BOSIACO `EntityDialog` and form section primitives because those areas do not yet expose standalone reusable creation dialogs.
- No persistence or service behavior was changed.

## Visual Design

- Quick Create results appear before navigation and records.
- Create actions use a distinct green accent, plus-style action language and a `Quick Create` badge.
- Navigation and record rows keep their existing blue/neutral command treatment.
- The main Command Center stays compact, keyboard-first and professional.

## Keyboard Workflow

- `⌘K` on Mac or `Ctrl+K` on Windows/Linux opens the Command Center.
- Typing filters Quick Create, Navigation and Records instantly.
- Arrow Down / Arrow Up navigates all visible results.
- Enter on a Quick Create result closes the palette and opens the dialog.
- Esc closes the palette.
- Tab remains free and is not trapped.

## Desktop Experience

The fastest path to start common work is now:

1. Open Command Center.
2. Type `new invoice`, `quote`, `contact` or similar.
3. Press Enter.
4. Continue in the creation dialog.

No sidebar or workspace navigation is required to begin the flow.

## Mobile Preparation

- No mobile-specific trigger was added.
- The Action Registry can be reused later by a floating mobile action button or other quick-entry surface.
