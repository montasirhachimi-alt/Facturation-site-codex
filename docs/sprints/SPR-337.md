# SPR-337 — Universal Command Center (Command Palette)

Date: 2026-07-09

## Objective

Create the first productivity-oriented Universal Command Center for BOSIACO. Version 1 is navigation-only: it opens major workspaces quickly from a global keyboard shortcut and local instant filtering.

## Scope

- UI/productivity layer only.
- Navigation commands only.
- No Runtime behavior, Services, APIs, Prisma, database, permission or business logic changes.
- No AI, workflow automation or backend search.

## Architecture

- The existing ERP-level `UniversalSearchProvider` remains the single global shell entry point.
- The placeholder global search sections now resolve to command-center navigation sections.
- Selection uses the local Next.js router and closes the palette automatically.
- The topbar trigger now clearly opens the command center while preserving the existing `⌘K` / `Ctrl+K` shortcut display.

## Command Registry

- Added a reusable `CommandCenterRegistry`.
- Commands register through `register` / `registerMany`.
- Navigation commands are composed outside the modal from:
  - seed workspace commands for Dashboard, Sales, HR and Reports
  - business module navigation metadata
  - existing sidebar navigation groups
- The modal does not hardcode its command list.
- Future versions can add providers for quick create, recent items, favorites, AI actions, workflow commands and global search.

## Supported Commands

Version 1 supports local navigation to:

- Dashboard
- CRM
- Companies
- Customers
- Contacts
- Activities
- Meetings
- Tasks
- Notes
- Sales
- Quotes
- Invoices
- Payments
- Products
- HR
- Reports
- Settings

Aliases support fast typing such as `quo`, `fact`, `pay`, `cli`, `crm`, `soc` and `cont`.

## Keyboard Experience

- `⌘K` on Mac opens the command center.
- `Ctrl+K` on Windows/Linux opens the command center.
- Focus automatically enters the input.
- Arrow Down and Arrow Up move through commands.
- Enter opens the selected workspace.
- Esc closes the palette.
- Tab is not trapped.

## Desktop Experience

- Compact command surface, designed for fast keyboard-first navigation.
- No oversized modal.
- Local filtering only, with no server requests.
- Navigation closes the palette automatically.

## Mobile Preparation

- No mobile shortcut behavior was added.
- The provider and registry remain reusable for a future floating mobile trigger.
