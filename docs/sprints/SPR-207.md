# SPR-207 — Widget Runtime Foundation

## Purpose

Create the invisible execution foundation for future HicoPilot dashboard widgets.

## Scope

- No dashboard redesign.
- No layout changes.
- No new widgets.
- No database or Prisma changes.
- No routing, sidebar or permission changes.

## Implementation

SPR-207 introduces a Widget Runtime that sits between the Dashboard and Workspace Context.

Architecture:

Dashboard
↓
Widget Runtime
↓
Workspace Context
↓
Workspace Service
↓
Core Engines

## Runtime Responsibilities

The runtime prepares shared widget context:

- current workspace
- workspace snapshot
- workspace preferences
- widget permissions foundation
- visibility state
- loading state
- error state
- pinned widget identifiers
- hidden widget identifiers
- refresh hooks

## Future Extension Points

- Widget Registry integration
- Widget permissions enforcement
- Hidden and pinned widgets
- Widget preferences
- Lazy loading
- Widget refresh
- Widget metadata
- Widget actions
- AI widgets
