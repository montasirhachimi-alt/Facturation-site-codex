# SPR-338 — Smart Global Search

Date: 2026-07-09

## Objective

Extend the Universal Command Center from navigation-only commands into Smart Global Search V1. The command center now searches both workspace destinations and existing local business records without backend search, AI, database changes or server requests.

## Scope

- Local/client search only.
- Existing demo/local data only.
- No Runtime, Services, APIs, Prisma, database, permission or business logic changes.
- No backend endpoints, server search, AI or workflow automation.

## Search Architecture

- Navigation commands remain in `CommandCenterRegistry`.
- Business record results live in a separate `RecordSearchRegistry`.
- `getFoundationSearchSections` composes both registries into grouped sections.
- The dialog remains generic and does not hardcode records.
- Results keep existing keyboard behavior and use `href` destinations for local routing.

## Data Sources Used

- `crmCompanySeed`
- `crmContactSeed`
- `crmCustomerSeed`
- `crmOpportunitySeed`
- `quoteSeed`
- `invoiceSeed`
- `paymentSeed`

No new business data was invented.

## Result Groups

- `Navigation`: workspace and module commands.
- `Records`: companies, contacts, customers, quotes, invoices, payments and opportunities.
- Suggested Actions remains intentionally unimplemented for a future sprint.

## Supported Record Types

- Companies route to company details.
- Contacts route to contact details.
- Customers route to the clients workspace.
- Quotes route to quote details.
- Invoices route to invoice details.
- Payments route to payment details.
- Opportunities route to the pipeline workspace.

## Result Design

Each result exposes:

- icon
- title
- type badge
- short metadata description
- route destination

## Keyboard Behavior

- `⌘K` / `Ctrl+K` opens the Command Center.
- Typing filters navigation and record results instantly.
- Arrow Up / Down navigates through all visible results.
- Enter opens the selected result and closes the palette.
- Esc closes the palette.
- Tab is not trapped.
